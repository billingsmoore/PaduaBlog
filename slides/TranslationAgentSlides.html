Study 1 — Easy Inputs
When baseline quality is already high, no augmentation moves the average.
All p-values > 0.05 · Baseline mean: 90.10 · Ceiling effect: insufficient headroom for improvement
Study 1 — Easy Inputs · Quality Band Distribution
Even on easy text, the glossary is the only tool that prevents catastrophic failures.
Baseline min score: 30 · Glossary min score: 78 · Glossary is the only condition with zero sub-75 outputs
Study 2 — Hard Inputs
On difficult text, every augmentation raises quality by ~26 points — a 38% improvement over baseline.
All augmented conditions: p < 0.0001 · Effect size ≈ 25 pts · Hard test set: sentences where baseline scored < 75
Study 2 — Hard Inputs · Quality Band Distribution
The glossary eliminates critical failures entirely. Combining all tools produces the most.
Baseline: 24 critical failures (<50) · Glossary only: 0 · All interventions: 3 — worse than any single tool
Study 2 — Hard Inputs · Mean vs. Variance
The glossary has the highest mean and lowest variance of any condition. Stacking all tools is the weakest approach.
Lower SD = more predictable quality · Glossary SD: 8.02 · All interventions SD: 13.63
Study 3 — Inference Scaling (Best-of-N)
Translating twice captures 58% of the total gain from translating five times.
N=1→2: +5.26 pts · N=2→3: +2.27 pts · N=3→5: +1.46 pts combined · All steps p < 0.003
Study 4 — Translation Memory Size
A single in-domain example is as effective as a corpus of 50,000.
Every TM size vs. baseline: p < 0.0001 · No adjacent TM size comparison reached significance · All TM conditions within a 1.6-pt band
Cross-Study Finding
Every strategy — tools, scaling, memory — hits the same ceiling near 82–84. The bottleneck is the base model.
Glossary (Study 2): 82.81 · Best-of-5 (Study 3): 84.47 · Best TM size (Study 4): 82.44 · Independent mechanisms, same ceiling
